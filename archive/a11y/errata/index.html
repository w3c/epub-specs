Errata for EPUB Accessibility 1.0

This page lists errata for the EPUB Accessibility 1.0 specification.

3.2 Package Metadata

The EPUB Accessibility 1.1 specification changes the requirement for an accessibilitySummary property to a recommendation.

For backwards compatibility, this erratum changes the same requirement in the 1.0 specification to a recommendation. EPUB creators are not required to include a summary to meet the discovery metadata conformance requirements.

It is also no longer necessary to repeat conformance information that is already expressed in the other accessibility metadata (e.g., conformance claims, hazards, and features) but this is not a normative requirement for conformance with the 1.0 specification.
